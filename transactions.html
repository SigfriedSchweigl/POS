JDBC – Aufgabe 7: Transaktionen

Mehrere Datenbankoperationen sicher zusammenfassen

Start JDBC Aufgabe 7

Ziel

Ziel ist es, mehrere Datenbankoperationen zu einer Transaktion zusammenzufassen. Dabei soll gezeigt werden, wie Änderungen entweder vollständig übernommen (commit()) oder bei Fehlern zurückgesetzt (rollback()) werden.

Aufgabe 7 – Transaktionen

Alles oder nichts – commit und rollback

Aufgabe:

Erstelle ein Java-Programm, das zwei Datenbankoperationen ausführt:

  • eine Person einfügen
  • eine andere Person löschen

Beide Operationen sollen Teil einer Transaktion sein.

Wenn ein Fehler auftritt, sollen alle Änderungen rückgängig gemacht werden.

Schritt 1: Verbindung herstellen

Hinweis

Verwende wie bisher die connection.properties-Datei.

Schritt 2: Auto-Commit deaktivieren

Deaktiviere das automatische Speichern von Änderungen.

Hinweis

conn.setAutoCommit(false);

Jetzt werden Änderungen erst mit commit() gespeichert.

Schritt 3: Erste Operation ausführen

Füge eine neue Person in die Tabelle ein.

Hinweis

INSERT INTO person (id, name, age) VALUES (?, ?, ?)

Schritt 4: Zweite Operation ausführen

Lösche eine bestehende Person aus der Tabelle.

Hinweis

DELETE FROM person WHERE id = ?

Schritt 5: Transaktion bestätigen

Übernimm alle Änderungen.

Hinweis

conn.commit();

Schritt 6: Fehler behandeln

Falls ein Fehler auftritt, sollen alle Änderungen zurückgesetzt werden.

Hinweis

conn.rollback();

Dadurch werden alle Änderungen der Transaktion rückgängig gemacht.

Schritt 7: Warum sind Transaktionen wichtig?

Hinweis

Transaktionen stellen sicher, dass mehrere Operationen nur gemeinsam ausgeführt werden.

Beispiel:

  • Geld wird abgebucht
  • Geld wird gutgeschrieben

👉 Beide Schritte müssen zusammen funktionieren.

Lösung

import java.sql.*;
import java.io.InputStream;
import java.nio.file.Files;
import java.nio.file.Path;
import java.util.Properties;

public class JdbcTransactionExample {

    public static void main(String[] args) {

        Properties props = new Properties();

        try (InputStream is = Files.newInputStream(Path.of("connection.properties"))) {

            props.load(is);

            String url = props.getProperty("jdbc_url");
            String user = props.getProperty("jdbc_user");
            String pwd = props.getProperty("jdbc_pwd");

            try (Connection conn = DriverManager.getConnection(url, user, pwd)) {

                conn.setAutoCommit(false);

                try {

                    String insertSql =
                            "INSERT INTO person (id, name, age) VALUES (?, ?, ?)";

                    try (PreparedStatement pstmt = conn.prepareStatement(insertSql)) {
                        pstmt.setInt(1, 5);
                        pstmt.setString(2, "Julia");
                        pstmt.setInt(3, 28);
                        pstmt.executeUpdate();
                    }

                    String deleteSql =
                            "DELETE FROM person WHERE id = ?";

                    try (PreparedStatement pstmt = conn.prepareStatement(deleteSql)) {
                        pstmt.setInt(1, 2);
                        pstmt.executeUpdate();
                    }

                    conn.commit();
                    System.out.println("Transaktion erfolgreich!");

                } catch (Exception e) {

                    conn.rollback();
                    System.out.println("Fehler! Änderungen wurden zurückgesetzt.");
                }

            }

        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
