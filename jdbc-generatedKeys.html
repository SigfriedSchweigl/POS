JDBC – Aufgabe 5: AutoGenerated Keys

ID automatisch erzeugen und auslesen

Start JDBC Aufgabe 5

Ziel

Ziel ist es, einen Datensatz in die Datenbank einzufügen, ohne die ID selbst zu setzen. Die Datenbank erzeugt die ID automatisch und diese wird anschließend ausgelesen.

Aufgabe 5 – Generated Keys

Automatisch generierte ID auslesen

Aufgabe:

Erstelle ein Java-Programm, das eine neue Person in die Tabelle person einfügt. Die ID soll dabei automatisch von der Datenbank erzeugt werden.

Lies anschließend die erzeugte ID aus und gib sie auf der Konsole aus.

Schritt 1: Tabelle anpassen

Passe die Tabelle so an, dass die ID automatisch generiert wird.

Hinweis

Verwende eine auto-generierte ID:

id IDENTITY PRIMARY KEY

Lösung: schema.sql

DROP TABLE IF EXISTS person;

CREATE TABLE person (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100),
    age INT
);

Schritt 2: INSERT ohne ID

Füge einen Datensatz ein, ohne die ID anzugeben.

Hinweis

Die SQL-Anweisung enthält keine ID: 

INSERT INTO person (name, age) VALUES (?, ?)

Schritt 3: Generated Keys aktivieren

Sorge dafür, dass die generierte ID zurückgegeben wird.

Hinweis

Verwende beim Erstellen des PreparedStatements: 

conn.prepareStatement(sql, Statement.RETURN_GENERATED_KEYS)

Schritt 4: Datensatz einfügen

Setze die Parameter und führe das INSERT aus.

Hinweis

pstmt.setString(1, "Tom");
pstmt.setInt(2, 30);

pstmt.executeUpdate();

Schritt 5: ID auslesen

Lies die automatisch generierte ID aus.

Hinweis

Verwende:

ResultSet rs = pstmt.getGeneratedKeys();

Und lies die ID:

if (rs.next()) {
    int id = rs.getInt(1);
}

Schritt 6: ID ausgeben

Gib die erzeugte ID auf der Konsole aus.

Lösung

import java.sql.*;
import java.io.InputStream;
import java.nio.file.Files;
import java.nio.file.Path;
import java.util.Properties;

public class JdbcGeneratedKeysExample {

    public static void main(String[] args) {

        Properties props = new Properties();

        try (InputStream is = Files.newInputStream(Path.of("connection.properties"))) {

            props.load(is);

            String url = props.getProperty("jdbc_url");
            String user = props.getProperty("jdbc_user");
            String pwd = props.getProperty("jdbc_pwd");

            try (Connection conn = DriverManager.getConnection(url, user, pwd)) {

                String sql = "INSERT INTO person (name, age) VALUES (?, ?)";

                try (PreparedStatement pstmt =
                             conn.prepareStatement(sql, Statement.RETURN_GENERATED_KEYS)) {

                    pstmt.setString(1, "Tom");
                    pstmt.setInt(2, 30);

                    pstmt.executeUpdate();

                    ResultSet rs = pstmt.getGeneratedKeys();

                    if (rs.next()) {
                        int generatedId = rs.getInt(1);
                        System.out.println("Generierte ID: " + generatedId);
                    }
                }

            }

        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
