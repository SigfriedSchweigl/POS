Aufgabe 8 – Architecture mit Repository

ActorApp mit Domain- und Persistence-Schicht

Aufgabe:

Erstelle ein Java-Programm mit folgender Struktur:

  • architecture.domain.Actor mit den Attributen id, firstName und lastName
  • architecture.persistence.ActorRepository als Interface
  • architecture.persistence.JdbcActorRepository als JDBC-Implementierung
  • architecture.app.ActorApp als Hauptklasse

Die Anwendung soll eine Verbindung zur H2-Datenbank herstellen und die Methoden des Repositories aufrufen.

Verwendet werden sollen:

  • findAll()
  • findAllByLastNameStartsWith(String prefix)
  • findById(Integer id)
  • save(Actor actor)

Voraussetzung: Die Datenbank wird über connection.properties aufgebaut, dabei wird das SQL-Skript schema.sql geladen.

Schritt 1: Projektstruktur anlegen

Lege zuerst die Packages für die drei Bereiche an:

  • architecture.domain
  • architecture.persistence
  • architecture.app

Hinweis

Die Idee dahinter ist, dass nicht alles in einer einzigen Klasse steht.

So wird der Code übersichtlicher:

  • domain → Fachobjekte
  • persistence → Datenbankzugriff
  • app → Programmstart

Lösung: mögliche Struktur

src
└── architecture
    ├── app
    │   └── ActorApp.java
    ├── domain
    │   └── Actor.java
    └── persistence
        ├── ActorRepository.java
        └── JdbcActorRepository.java

Schritt 4: Domain-Klasse Actor erstellen

Erstelle nun die Klasse Actor im Package architecture.domain.

Hinweis

Die Klasse soll die Attribute id, firstName und lastName enthalten.

Ergänze außerdem Konstruktor, Getter und eine sinnvolle toString()-Methode.

Lösung: Actor.java

package architecture.domain;

public class Actor {
    private Integer id;
    private String firstName;
    private String lastName;

    public Actor(Integer id, String firstName, String lastName) {
        this.id = id;
        this.firstName = firstName;
        this.lastName = lastName;
    }

    public Integer getId() {
        return id;
    }

    public String getFirstName() {
        return firstName;
    }

    public String getLastName() {
        return lastName;
    }

    @Override
    public String toString() {
        return id + " | " + firstName + " | " + lastName;
    }
}

Schritt 5: Repository-Interface definieren

Erstelle nun das Interface ActorRepository im Package architecture.persistence.

Hinweis

Das Interface beschreibt nur, was möglich sein soll, aber noch nicht, wie es umgesetzt wird.

Verwende die Methoden:

  • findAll()
  • findAllByLastNameStartsWith(String prefix)
  • findById(Integer id)
  • save(Actor actor)

Lösung: ActorRepository.java

package architecture.persistence;

import architecture.domain.Actor;

import java.sql.SQLException;
import java.util.List;

public interface ActorRepository {

Schritt 12: Methoden des Repositories aufrufen

Rufe nun alle Methoden auf und gib die Ergebnisse auf der Konsole aus.

Hinweis

Teste zum Beispiel in dieser Reihenfolge:

  • findAll()
  • findAllByLastNameStartsWith("J")
  • findById(2)
  • save(...)
  • danach nochmals findAll()

Erklärung

Durch die Aufteilung in domain, persistence und app wird der Code klarer strukturiert.

  • Actor beschreibt das Fachobjekt
  • ActorRepository beschreibt die verfügbaren Operationen
  • JdbcActorRepository enthält den konkreten JDBC-Code
  • ActorApp startet das Programm

Dadurch ist der Code leichter zu verstehen, zu testen und später zu erweitern.

Erklärung

Das Interface trennt die Beschreibung der Methoden von der eigentlichen Umsetzung.

Später könnte man zum Beispiel statt JdbcActorRepository auch eine andere Implementierung schreiben, ohne die App zu ändern.

Lösung: ActorRepository.java

package architecture.persistence;

import architecture.domain.Actor;

import java.sql.SQLException;
import java.util.List;
import java.util.Optional;

public interface ActorRepository {
    List findAll() throws SQLException;

    List findAllByLastNameStartsWith(String prefix) throws SQLException;

    Optional findById(Integer id) throws SQLException;

    Actor save(Actor actor) throws SQLException;
}

Lösung: JdbcActorRepository.java

package architecture.persistence;

import architecture.domain.Actor;

import java.sql.Connection;
import java.sql.PreparedStatement;
import java.sql.ResultSet;
import java.sql.SQLException;
import java.util.ArrayList;
import java.util.List;
import java.util.Optional;

public class JdbcActorRepository implements ActorRepository {

    private final Connection conn;

    public JdbcActorRepository(Connection conn) {
        this.conn = conn;
    }

    @Override
    public List findAll() throws SQLException {
        String sql = "SELECT * FROM actor ORDER BY act_id";
        List result = new ArrayList<>();

        try (PreparedStatement pstmt = conn.prepareStatement(sql);
             ResultSet rs = pstmt.executeQuery()) {

            while (rs.next()) {
                result.add(mapRow(rs));
            }
        }

        return result;
    }

    @Override
    public List findAllByLastNameStartsWith(String prefix) throws SQLException {
        String sql = """
                SELECT * FROM actor
                WHERE act_lastname LIKE ?
                ORDER BY act_id
                """;

        List result = new ArrayList<>();

        try (PreparedStatement pstmt = conn.prepareStatement(sql)) {
            pstmt.setString(1, prefix + "%");

            try (ResultSet rs = pstmt.executeQuery()) {
                while (rs.next()) {
                    result.add(mapRow(rs));
                }
            }
        }

        return result;
    }

    @Override
    public Optional findById(Integer id) throws SQLException {
        String sql = "SELECT * FROM actor WHERE act_id = ?";

        try (PreparedStatement pstmt = conn.prepareStatement(sql)) {
            pstmt.setInt(1, id);

            try (ResultSet rs = pstmt.executeQuery()) {
                if (rs.next()) {
                    return Optional.of(mapRow(rs));
                }
            }
        }

        return Optional.empty();
    }

    @Override
    public Actor save(Actor actor) throws SQLException {
        String sql = """
                INSERT INTO actor (act_id, act_firstname, act_lastname)
                VALUES (?, ?, ?)
                """;

        try (PreparedStatement pstmt = conn.prepareStatement(sql)) {
            pstmt.setInt(1, actor.getId());
            pstmt.setString(2, actor.getFirstName());
            pstmt.setString(3, actor.getLastName());
            pstmt.executeUpdate();
        }

        return actor;
    }

    private Actor mapRow(ResultSet rs) throws SQLException {
        Integer id = rs.getInt("act_id");
        String firstName = rs.getString("act_firstname");
        String lastName = rs.getString("act_lastname");
        return new Actor(id, firstName, lastName);
    }
}

Lösung: ActorApp.java

package architecture.app;

import architecture.domain.Actor;
import architecture.persistence.ActorRepository;
import architecture.persistence.JdbcActorRepository;

import java.io.InputStream;
import java.nio.file.Files;
import java.nio.file.Path;
import java.sql.Connection;
import java.sql.DriverManager;
import java.util.Properties;

public class ActorApp {

    public static void main(String[] args) {

        Properties props = new Properties();
        Path path = Path.of("connection.properties");

        if (!Files.exists(path)) {
            System.out.println("Properties-Datei nicht gefunden!");
            return;
        }

        try (InputStream is = Files.newInputStream(path)) {

            props.load(is);

            String url = props.getProperty("jdbc_url");
            String user = props.getProperty("jdbc_user");
            String pwd = props.getProperty("jdbc_pwd");

            try (Connection conn = DriverManager.getConnection(url, user, pwd)) {

                ActorRepository repo = new JdbcActorRepository(conn);

                System.out.println("=== Alle Schauspieler ===");
                for (Actor actor : repo.findAll()) {
                    System.out.println(actor);
                }

                System.out.println();
                System.out.println("=== Nachname beginnt mit J ===");
                for (Actor actor : repo.findAllByLastNameStartsWith("J")) {
                    System.out.println(actor);
                }

                System.out.println();
                System.out.println("=== Schauspieler mit ID 2 ===");
                System.out.println(repo.findById(2).orElse(null));

                System.out.println();
                System.out.println("=== Neuer Schauspieler wird gespeichert ===");
                Actor newActor = new Actor(5, "Jennifer", "Lawrence");
                repo.save(newActor);
                System.out.println(newActor);

                System.out.println();
                System.out.println("=== Alle Schauspieler nach save() ===");
                for (Actor actor : repo.findAll()) {
                    System.out.println(actor);
                }

            } catch (Exception e) {
                System.out.println("Fehler bei der Datenbankoperation:");
                e.printStackTrace();
            }

        } catch (Exception e) {
            System.out.println("Fehler beim Laden der Properties:");
            e.printStackTrace();
        }
    }
}
