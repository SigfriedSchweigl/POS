JDBC – Aufgabe 1: Datenbankverbindung testen

Schrittweise Verbindung zu H2 und SQL Server mit Hinweisen

Start JDBC H2 SQL Server

Ziel

Ziel ist es, eine Datenbankverbindung aufzubauen und Informationen über die Datenbank mit DatabaseMetaData auszulesen.

Aufgabe 1A – H2

Connection zur H2-Datenbank

Aufgabe:

Stelle eine Verbindung zu einer H2-Datenbank her und gib Informationen über die Datenbank aus.

Schritt 1: Maven Dependency hinzufügen

Füge den H2-Treiber in deine pom.xml ein.

Hinweis

Dependencies findest du auf der offiziellen Maven-Seite:

Maven Repository 

<dependency>
  <groupId>com.h2database</groupId>
  <artifactId>h2</artifactId>
  <version>2.2.224</version>
</dependency>

Schritt 2: Verbindung definieren

Verwende folgende Daten:

jdbc:h2:~/test
User: sa
Passwort:

Hinweis

Die URL beginnt bei H2 immer mit jdbc:h2:

Schritt 3: Verbindung herstellen

Nutze JDBC, um eine Verbindung aufzubauen.

Hinweis

Verwende DriverManager.getConnection(...)

Schritt 4: Datenbankinformationen ausgeben

Gib Informationen über die Datenbank aus.

Hinweis

Nutze connection.getMetaData()

Lösung

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.DatabaseMetaData;
import java.sql.SQLException;

public class H2Test {
    public static void main(String[] args) {

        String url = "jdbc:h2:~/test";
        String user = "sa";
        String password = "";

        try (Connection connection = DriverManager.getConnection(url, user, password)) {

            System.out.println("Verbunden mit Datenbank...");

            DatabaseMetaData dbmd = connection.getMetaData();

            System.out.println("dbmd:driver version = " + dbmd.getDriverVersion());
            System.out.println("dbmd:driver name = " + dbmd.getDriverName());
            System.out.println("db name = " + dbmd.getDatabaseProductName());
            System.out.println("db ver = " + dbmd.getDatabaseProductVersion());

        } catch (SQLException e) {
            e.printStackTrace();
        }
    }
}
Aufgabe 1B – SQL Server

Connection zu SQL Server

Aufgabe:

Stelle eine Verbindung zu einem SQL Server her und gib ebenfalls die Metadaten aus.

Schritt 1: Maven Dependency

Hinweis

Suche nach: mssql-jdbc auf mvnrepository.com

Schritt 2: Verbindungsdaten

jdbc:sqlserver://localhost:1433;databaseName=School;encrypt=true;trustServerCertificate=true

Hinweis

Achte darauf, dass dein SQL Server läuft.

Schritt 3: Verbindung herstellen

Hinweis

Gleiche Vorgehensweise wie bei H2.

Lösung

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.DatabaseMetaData;
import java.sql.SQLException;

public class SqlTest {
    public static void main(String[] args) {

        String url = "jdbc:sqlserver://localhost:1433;databaseName=School;encrypt=true;trustServerCertificate=true";
        String user = "sa";
        String password = "DeinPasswort";

        try (Connection connection = DriverManager.getConnection(url, user, password)) {

            System.out.println("Verbunden mit Datenbank...");

            DatabaseMetaData dbmd = connection.getMetaData();

            System.out.println("dbmd:driver version = " + dbmd.getDriverVersion());
            System.out.println("dbmd:driver name = " + dbmd.getDriverName());
            System.out.println("db name = " + dbmd.getDatabaseProductName());
            System.out.println("db ver = " + dbmd.getDatabaseProductVersion());

        } catch (SQLException e) {
            e.printStackTrace();
        }
    }
}
