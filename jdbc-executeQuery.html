Ziel

Ziel ist es, eine Verbindung zu einer H2-Datenbank über eine connection.properties-Datei herzustellen, automatisch ein SQL-Skript auszuführen und anschließend mit Statement und executeQuery() Daten aus einer Tabelle auszulesen.

Aufgabe 2 – H2 mit Statement

Daten mit Statement und executeQuery auslesen

Aufgabe:

Erstelle ein Java-Programm, das die Verbindungsdaten aus einer connection.properties-Datei lädt und damit eine Verbindung zu einer H2-Datenbank herstellt.

Über die JDBC-URL soll beim Verbindungsaufbau automatisch ein SQL-Skript ausgeführt werden. Dieses Skript soll eine Tabelle erstellen und Beispieldaten einfügen.

Lies anschließend mit einem Statement und executeQuery() alle Datensätze aus der Tabelle aus und gib sie auf der Konsole aus.

Schritt 1: Properties-Datei erstellen

Lege eine Datei connection.properties an und speichere darin die Verbindungsdaten für H2.

Hinweis

Verwende in der JDBC-URL zusätzlich INIT=RUNSCRIPT FROM 'schema.sql'.

Dadurch wird beim Aufbau der Verbindung automatisch das SQL-Skript ausgeführt.

Lösung: connection.properties

jdbc_url=jdbc:h2:mem:testdb;INIT=RUNSCRIPT FROM 'schema.sql'
jdbc_user=sa
jdbc_pwd=

Schritt 2: SQL-Datei erstellen

Lege eine Datei schema.sql an. Diese Datei soll eine Tabelle erstellen und Beispieldaten einfügen.

Hinweis

Erstelle zum Beispiel eine Tabelle person mit den Spalten id, name und age.

Lösung: schema.sql

DROP TABLE IF EXISTS person;

CREATE TABLE person (
    id INT PRIMARY KEY,
    name VARCHAR(100),
    age INT
);

INSERT INTO person VALUES (1, 'Anna', 22);
INSERT INTO person VALUES (2, 'Max', 25);
INSERT INTO person VALUES (3, 'Lisa', 19);

Schritt 3: Properties-Datei im Java-Programm laden

Lade die Datei connection.properties mit einem Properties-Objekt.

Hinweis

Verwende Files.newInputStream(Path.of("connection.properties")).

Schritt 4: Verbindung herstellen

Lies die Werte aus der Properties-Datei mit getProperty(...) aus und verwende sie für DriverManager.getConnection(...).

Hinweis

Du brauchst folgende Parameter für getProperty(...):

  • jdbc_url
  • jdbc_user
  • jdbc_pwd

Schritt 5: SELECT-Abfrage mit Statement ausführen

Erstelle ein Statement und führe mit executeQuery() eine SELECT-Abfrage auf der Tabelle person aus.

Hinweis

Erstelle zuerst ein Statement-Objekt: 

Statement stmt = conn.createStatement();

Führe anschließend eine SQL-Abfrage aus: 

SELECT * FROM person

Verwende dafür stmt.executeQuery(...).

Schritt 6: Ergebnisse ausgeben

Durchlaufe das ResultSet und gib alle Datensätze auf der Konsole aus.

Hinweis

Verwende eine Schleife mit while (rs.next()).

Die Spalten kannst du mit getInt(...) und getString(...) auslesen.

Lösung

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.ResultSet;
import java.sql.Statement;
import java.sql.SQLException;
import java.io.InputStream;
import java.nio.file.Files;
import java.nio.file.Path;
import java.util.Properties;

public class JdbcStatementExample {

    public static void main(String[] args) {

        Properties props = new Properties();

        Path path = Path.of("connection.properties");

        if (!Files.exists(path)) {
            System.out.println("Properties-Datei nicht gefunden!");
            return;
        }

        try (InputStream is = Files.newInputStream(path)) {

            props.load(is);

            String url = props.getProperty("jdbc_url");
            String user = props.getProperty("jdbc_user");
            String pwd = props.getProperty("jdbc_pwd");

            try (Connection conn = DriverManager.getConnection(url, user, pwd);
                 Statement stmt = conn.createStatement()) {

                System.out.println("Verbindung erfolgreich!");

                String sql = "SELECT * FROM person";

                ResultSet rs = stmt.executeQuery(sql);

                while (rs.next()) {
                    int id = rs.getInt("id");
                    String name = rs.getString("name");
                    int age = rs.getInt("age");

                    System.out.println(id + " | " + name + " | " + age);
                }

            } catch (SQLException e) {
                System.out.println("Fehler bei der Datenbankverbindung:");
                e.printStackTrace();
            }

        } catch (Exception e) {
            System.out.println("Fehler beim Laden der Properties:");
            e.printStackTrace();
        }
    }
}
