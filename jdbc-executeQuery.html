JDBC – Aufgabe 2: Statement und executeQuery

Eine SQL-Datei automatisch ausführen und Daten mit executeQuery auslesen

Start JDBC Properties SQL-Datei Java

Ziel

Ziel ist es, eine JDBC-Verbindung über eine Properties-Datei herzustellen, beim Start automatisch ein SQL-Skript auszuführen und anschließend Daten mit Statement und executeQuery() aus einer Tabelle auszulesen.

Aufgabe 2

Statement und executeQuery mit H2

Aufgabe:

Erstelle ein Java-Programm, das eine Verbindung zu einer H2-Datenbank herstellt. Die Verbindungsdaten sollen aus einer Properties-Datei geladen werden. Über die JDBC-URL soll beim Verbindungsaufbau automatisch ein SQL-Skript ausgeführt werden, das eine Tabelle erstellt und Beispieldaten einfügt.

Lies danach mit einem Statement und executeQuery() alle Datensätze aus der Tabelle aus und gib sie auf der Konsole aus.

Teilaufgaben

  • Eine connection.properties-Datei anlegen
  • Eine schema.sql-Datei erstellen
  • Ein Java-Programm schreiben, das die Properties lädt
  • Eine Verbindung zur H2-Datenbank aufbauen
  • Mit Statement eine SELECT-Abfrage ausführen
  • Die Daten mit ResultSet ausgeben
Schritt 1

Properties-Datei erstellen

Lege eine Datei mit dem Namen connection.properties an. In dieser Datei sollen die Verbindungsdaten für die H2-Datenbank stehen.

Hinweis

Die JDBC-URL enthält zusätzlich den Parameter INIT=RUNSCRIPT FROM 'schema.sql'.

Dadurch wird beim Aufbau der Verbindung automatisch die Datei schema.sql ausgeführt.

Lösung: connection.properties

jdbc_url=jdbc:h2:mem:testdb;INIT=RUNSCRIPT FROM 'schema.sql'
jdbc_user=sa
jdbc_pwd=
Schritt 2

SQL-Datei erstellen

Lege eine Datei mit dem Namen schema.sql an. Diese Datei soll:

  • eine Tabelle erstellen
  • mehrere Datensätze einfügen

Dadurch stehen nach dem Verbindungsaufbau sofort Daten zum Auslesen bereit.

Hinweis

Verwende zum Beispiel eine Tabelle person mit den Spalten:

  • id
  • name
  • age

Lösung: schema.sql

DROP TABLE IF EXISTS person;

CREATE TABLE person (
    id INT PRIMARY KEY,
    name VARCHAR(100),
    age INT
);

INSERT INTO person VALUES (1, 'Anna', 22);
INSERT INTO person VALUES (2, 'Max', 25);
INSERT INTO person VALUES (3, 'Lisa', 19);
Schritt 3

Properties-Datei im Java-Programm laden

Lade die Datei connection.properties mit Hilfe eines InputStream und eines Properties-Objekts.

Hinweis

Du kannst dafür Files.newInputStream(Path.of(...)) verwenden.

Schritt 4

Verbindung aufbauen

Lies die Werte aus der Properties-Datei aus und verwende sie für DriverManager.getConnection(...).

Hinweis

Du brauchst die drei Werte:

  • jdbc_url
  • jdbc_user
  • jdbc_pwd
Schritt 5

SELECT-Abfrage mit Statement ausführen

Erstelle ein Statement und führe mit executeQuery() eine Abfrage auf der Tabelle person aus.

Hinweis

Verwende eine SQL-Abfrage wie: 

SELECT * FROM person
Schritt 6

Ergebnisse ausgeben

Durchlaufe das ResultSet und gib alle Datensätze auf der Konsole aus.

Hinweis

Verwende eine while (rs.next())-Schleife.

Die Werte kannst du z. B. mit getInt(...) und getString(...) auslesen.

Gesamtlösung

Java-Code

Lösung: JdbcStatementExample.java

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.ResultSet;
import java.sql.Statement;
import java.sql.SQLException;
import java.io.InputStream;
import java.nio.file.Files;
import java.nio.file.Path;
import java.util.Properties;

public class JdbcStatementExample {

    public static void main(String[] args) {

        Properties props = new Properties();

        Path path = Path.of("connection.properties");

        if (!Files.exists(path)) {
            System.out.println("Properties-Datei nicht gefunden!");
            return;
        }

        try (InputStream is = Files.newInputStream(path)) {

            props.load(is);

            String url = props.getProperty("jdbc_url");
            String user = props.getProperty("jdbc_user");
            String pwd = props.getProperty("jdbc_pwd");

            try (Connection conn = DriverManager.getConnection(url, user, pwd);
                 Statement stmt = conn.createStatement()) {

                System.out.println("Verbindung erfolgreich!");

                String sql = "SELECT * FROM person";

                ResultSet rs = stmt.executeQuery(sql);

                while (rs.next()) {
                    int id = rs.getInt("id");
                    String name = rs.getString("name");
                    int age = rs.getInt("age");

                    System.out.println(id + " | " + name + " | " + age);
                }

            } catch (SQLException e) {
                System.out.println("Fehler bei der Datenbankverbindung:");
                e.printStackTrace();
            }

        } catch (Exception e) {
            System.out.println("Fehler beim Laden der Properties:");
            e.printStackTrace();
        }
    }
}
Zusatz

Was wird hier gelernt?

  • Properties aus einer Datei laden
  • JDBC-Verbindungsdaten auslagern
  • Ein SQL-Skript automatisch ausführen
  • Statement verwenden
  • executeQuery() für SELECT-Abfragen einsetzen
  • Mit ResultSet Datensätze durchlaufen
