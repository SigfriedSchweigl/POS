Ziel

Ziel ist es, mit Statement und executeUpdate() SQL-Anweisungen auszuführen, die Daten in einer Tabelle verändern. Dabei soll auch der Rückgabewert von executeUpdate() ausgewertet werden.

Übung 3 – H2 mit executeUpdate

Daten einfügen, ändern und löschen

Übung:

Erstelle ein Java-Programm, das die Verbindungsdaten aus einer connection.properties-Datei lädt und damit eine Verbindung zu einer H2-Datenbank herstellt.

Über die JDBC-URL soll beim Verbindungsaufbau automatisch ein SQL-Skript ausgeführt werden. Dieses Skript soll eine Tabelle erstellen und Beispieldaten einfügen.

Führe anschließend mit einem Statement und executeUpdate() nacheinander folgende Aktionen aus:

  • einen neuen Datensatz einfügen
  • einen bestehenden Datensatz ändern
  • einen Datensatz löschen

Gib nach jeder Anweisung aus, wie viele Zeilen geändert wurden.

Schritt 1: Properties-Datei verwenden

Verwende wieder eine Datei connection.properties mit den Verbindungsdaten für H2.

Hinweis

Du kannst dieselbe Properties-Datei wie in Übung 2 verwenden.

Lösung: connection.properties

jdbc_url=jdbc:h2:mem:testdb;INIT=RUNSCRIPT FROM 'schema.sql'
jdbc_user=sa
jdbc_pwd=

Schritt 2: SQL-Datei erstellen

Lege eine Datei schema.sql an. Diese Datei soll eine Tabelle person erstellen und einige Beispieldaten einfügen.

Hinweis

Verwende wieder die Spalten id, name und age.

Lösung: schema.sql

DROP TABLE IF EXISTS person;

CREATE TABLE person (
    id INT PRIMARY KEY,
    name VARCHAR(100),
    age INT
);

INSERT INTO person VALUES (1, 'Anna', 22);
INSERT INTO person VALUES (2, 'Max', 25);
INSERT INTO person VALUES (3, 'Lisa', 19);

Schritt 3: Verbindung herstellen

Lade die Properties-Datei und stelle eine Verbindung zur Datenbank her.

Hinweis

Verwende wieder Files.newInputStream(Path.of("connection.properties")) und DriverManager.getConnection(...).

Schritt 4: Statement erstellen

Erstelle ein Statement-Objekt, mit dem du SQL-Anweisungen ausführen kannst.

Hinweis

Verwende: 

Statement stmt = conn.createStatement();

Schritt 5: Datensatz einfügen

Füge mit executeUpdate() einen neuen Datensatz in die Tabelle person ein.

Hinweis

Eine mögliche SQL-Anweisung ist: 

INSERT INTO person VALUES (4, 'Tom', 30)

executeUpdate() liefert die Anzahl der geänderten Zeilen zurück.

Schritt 6: Datensatz ändern

Ändere mit executeUpdate() das Alter einer Person.

Hinweis

Eine mögliche SQL-Anweisung ist: 

UPDATE person SET age = 26 WHERE id = 2

Schritt 7: Datensatz löschen

Lösche mit executeUpdate() einen Datensatz aus der Tabelle.

Hinweis

Eine mögliche SQL-Anweisung ist: 

DELETE FROM person WHERE id = 1

Schritt 8: Rückgabewert ausgeben

Gib nach jeder SQL-Anweisung aus, wie viele Zeilen geändert wurden.

Hinweis

Speichere den Rückgabewert von executeUpdate() in einer Variablen vom Typ int.

Lösung

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.Statement;
import java.sql.SQLException;
import java.io.InputStream;
import java.nio.file.Files;
import java.nio.file.Path;
import java.util.Properties;

public class JdbcExecuteUpdateExample {

    public static void main(String[] args) {

        Properties props = new Properties();
        Path path = Path.of("connection.properties");

        if (!Files.exists(path)) {
            System.out.println("Properties-Datei nicht gefunden!");
            return;
        }

        try (InputStream is = Files.newInputStream(path)) {

            props.load(is);

            String url = props.getProperty("jdbc_url");
            String user = props.getProperty("jdbc_user");
            String pwd = props.getProperty("jdbc_pwd");

            try (Connection conn = DriverManager.getConnection(url, user, pwd);
                 Statement stmt = conn.createStatement()) {

                System.out.println("Verbindung erfolgreich!");

                int insertedRows = stmt.executeUpdate(
                        "INSERT INTO person VALUES (4, 'Tom', 30)");
                System.out.println("Eingefügte Zeilen: " + insertedRows);

                int updatedRows = stmt.executeUpdate(
                        "UPDATE person SET age = 26 WHERE id = 2");
                System.out.println("Geänderte Zeilen: " + updatedRows);

                int deletedRows = stmt.executeUpdate(
                        "DELETE FROM person WHERE id = 1");
                System.out.println("Gelöschte Zeilen: " + deletedRows);

            } catch (SQLException e) {
                System.out.println("Fehler bei der Datenbankoperation:");
                e.printStackTrace();
            }

        } catch (Exception e) {
            System.out.println("Fehler beim Laden der Properties:");
            e.printStackTrace();
        }
    }
}
